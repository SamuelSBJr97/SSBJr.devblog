# ⚡ 30 Segundos para Começar

Não tem tempo? Aqui está tudo que precisa fazer:

## Step 1: Setup (10s)

```bash
npm install
```

## Step 2: Seu primeiro post (5s)

```bash
npm run new-post -- "Meu Primeiro Post"
```

Edite o arquivo criado em `src/posts/`.

## Step 3: Preview (5s)

```bash
npm run dev
```

Abra `dist/index.html` no navegador.

## Step 4: Deploy (10s)

```bash
git add .
git commit -m "Novo post"
git push
```

GitHub Actions faz o resto! 🎉
