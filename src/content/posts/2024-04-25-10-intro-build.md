# Introdução ao Build.js

O `build.js` é o coração do nosso gerador de blog estático. Ele converte arquivos Markdown em páginas HTML otimizadas.

## Como Funciona

1. Lê arquivos Markdown da pasta `src/content/`
2. Processa o front-matter (metadados)
3. Converte Markdown para HTML
4. Aplica templates
5. Gera arquivos na pasta `dist/`

## Exemplo de Uso

```javascript
npm run build
```

## Estrutura de Pastas

A estrutura segue o padrão de data:

```
src/content/
├── posts/
│   ├── 2024-04-25/
│   │   ├── 10-primeiro-post.md
│   │   └── 14-segundo-post.md
```
