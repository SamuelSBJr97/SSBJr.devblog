# E-commerce com Node.js

Um projeto completo de plataforma de e-commerce construída com tecnologias modernas.

## Stack Tecnológico

- **Backend**: Node.js + Express
- **Database**: PostgreSQL
- **Frontend**: React + Redux
- **Deploy**: Docker + AWS

## Funcionalidades

- 🛒 Carrinho de compras
- 💳 Integração com Stripe
- 📦 Gerenciamento de estoque
- 👤 Sistema de usuários
- 📊 Dashboard administrativo
